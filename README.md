# Exercicio_MemeToNervoso
Peça ao usuário que diga o que pensa sobre o programa. Pegue a resposta, mantenha só os primeiros 10 caracteres e descarte o resto. Depois exiba no formato meme:  Usuário, o que você pensa sobre mim? [resposta]  E sabe o que eu penso de você? [você é mui... + resposta cortada]... NÃO, PERA. Tô nervoso.
